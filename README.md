# Green Heritage CHS Portal

Simple beginner-friendly website for Green Heritage CHS, Kharghar.

## Features

- Resident login and signup
- Complaint submission with flat number and category
- Complaint status tracking: Pending, In Progress, Completed
- Resident feedback after completion
- Admin dashboard for complaint management
- Daily checklist management

## Demo Admin Login

- Email: `admin@greenheritage.in`
- Password: `admin123`

## Run

Open `index.html` in a browser.

All data is stored in browser `localStorage`, so it works without a backend.
